# PROJECT CONTEXT

Hệ thống ERP nội bộ cho doanh nghiệp thang máy khoảng 100 người. Trục nghiệp vụ chính:

Đăng ký khách hàng → Lịch chăm sóc → Báo giá → Hợp đồng → Dự án → Hồ sơ thang máy → Thi công → KCS → Kiểm định → Nghiệm thu → Bàn giao → Bảo hành → Bảo trì.

Không sử dụng khái niệm “Cơ hội bán hàng”. Dự án chỉ được tạo sau khi hợp đồng đã ký hoặc được phê duyệt triển khai.
